# Movie Watchlist Manager

A Python Tkinter application that allows users to create and manage a personal movie watchlist.

## Description
This application provides a simple graphical interface where users can add movies, assign genres, and manage their watchlist. The watchlist can be saved to a file and loaded later.

## Features
- Add movies with a selected genre
- View watchlist in a separate window
- Delete selected movies
- Clear the entire watchlist
- Save watchlist to a file
- Load watchlist from a file
- Scrollable table display

## How to Run
1. Make sure Python is installed
2. Run the program:
```bash
python MovieWatchlist_V6.py
